⚙ Connect Google Sheets
Your data will be stored in a Google Sheet in your own Drive. You only need to do this once.

Step-by-step setup:
1
Go to console.cloud.google.com → Create a new project (name it anything).
2
In the search bar type Google Sheets API → Click it → Click Enable.
3
Go to APIs & Services → Credentials → Create Credentials → API Key. Copy the key.
4
Go to sheets.google.com → Create a blank sheet → Copy the long ID from the URL (between /d/ and /edit).
5
Make the sheet publicly viewable: Share → Anyone with the link → Viewer. (The API key handles write access.)

Solo Queue — Questions Every Game
Pre-drop: Where did the hot drop go? Is my landing ring-viable if I take an immediate fight?
Mid game: Am I ahead of the ring? Is this fight worth taking? (Resource advantage + position advantage + exit = 2 of 3 minimum)
In the fight: Am I using cover as a transition tool? Where's the third party coming from? (Budget 30-45 sec)
Final 3 rings: Where do I want to be — can I get there now? Am I playing for placement or kills?
60-Second Rule
Before every requeue: Why did I die? Did I take a bad fight? Did I know where the ring was going before I had to move?
